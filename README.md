# 🌤️ Weather App

A clean, responsive weather application built with React and Vite. Search for real-time weather data by city, with a beautiful dark atmospheric UI.

🔗 **Live Demo:** [weather-app-co.netlify.app](https://weather-app-co.netlify.app)

---

## ✨ Features

- 🔍 Search weather by city name
- 🏙️ Quick-select dropdown with preset Bangladeshi cities
- 🌡️ Displays temperature, humidity, wind speed, visibility, sunrise & sunset
- ⚡ Demo mode with mock data (no API key required)
- 📱 Fully responsive — works on mobile, tablet, and desktop
- 🎨 Dynamic background that changes based on weather conditions

---

## 🛠️ Tech Stack

- **React** — UI components
- **Vite** — build tool and dev server
- **OpenWeatherMap API** — live weather data
- **CSS** — custom styling with animations

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/weather-app.git
cd weather-app
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```

Open **http://localhost:5173** in your browser.
